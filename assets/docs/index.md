# Unsplash Viewer

Sidebar CMS extension for browsing Unsplash, viewing photo metadata, and
copying image URLs into content.

## Setup

1. Install the app on your CMS instance.
2. Open the app settings and paste an Unsplash Access Key
   (https://unsplash.com/developers).
3. Open the CMS sidebar — the **Unsplash Viewer** panel appears.

If no per-install key is set, the app falls back to the
`APP_ENV_UNSPLASH_ACCESS_KEY` environment variable.
